# Hand Gesture Mouse Controller

A Python-based hand tracking system that controls mouse movement and clicks
based on finger gestures using OpenCV, MediaPipe, and PyAutoGUI.

## Features
- Cursor movement via index finger
- Left click (Index + Middle pinch)
- Right click (Middle + Ring pinch)
- Drag and Drop (Thumb + Index hold)
- Scroll with Index finger up/down
- Smooth cursor movement

## Run Instructions
pip install -r requirements.txt
python main.py
